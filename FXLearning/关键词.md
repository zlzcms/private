点大小：市场波动的最小单位

点值：波动一个点的价值

返佣要转成USD去返, 用品种结算的币种转USD

**IB返佣公式 ：**

**按点差返佣：**

返佣金额（品种结算币种） = 点差百分比* 点差* 点值价格(交易品种币种) * 收益组成

汇率转换： 利润（品种结算币种） x (汇率) = 返佣金额 (USD) USD在后面就是*

点值价格(交易品种币种)= 点大小 * 合约大小 * 交易量(手数)

点差= 开仓点差*0.5 + 平仓点差*0.5）*点大小

**每手固定返佣：**

返佣金额（USD）= 每手固定值 * 手数

收益组成 = （净值-赠金)/净值

部分数据信息：

返佣平仓汇率表 : t_ib_commission_logs 字段：symbol_profit_currency_rate

点差表：mt_deals 字段：（MarketAsk-MarketBid）

**多级返佣计算公式：**

**每手/金额 ：**

返佣金额（USD） = 手数 * (佣金差)

**点值 / 每手 /金额：**

返佣金额= 点值 * 手数 * 佣金差

点值 =（ 点大小 * 合约大小） / 汇率 USD在前面就是/

**百万交易量（开仓和平仓的手数都要计算）**

**外汇 ：**（ 开仓手数* contract-size（合同大小-交易端核对） * 收益组成））转USD / 100万 （ 平仓手数* contract-size（合同大小-交易端核对 * 收益组成））转USD / 100万

**非外汇（石油，股票，重金属）：**（ 开仓手数* contract-size（合同大小-交易端核对 ）* 收益组成）* price ）转USD / 100万 +（ 平仓手数* contract-size（合同大小-交易端核对） * 收益组成）* price ）转USD / 100万


**赠金的规则：** **1. 赠金配置：**什么账户类型---什么货币组--哪个IB下面的客户---才赠送赠金，按照IB来配置， **2. 赠金触发：** 2.1 真实的MT4, MT5 交易账号真实入金，INV入金不触发赠金 2.2 内部划转，外部划转，社交分润，pamm分润，IB向下返佣，IB向上返佣，多级返佣到账都不触发赠金 **3. 赠金扣除：** 3.1 出金提交立马清空赠金（当前操作的MT账号的赠金），出金取消和审核拒绝都不退回 3.2 内部MT划转，转出的MT账号赠金清空，转入的MT账号不触发赠金 3.3 持仓中且开市时，MT交易账号净值 小于等于 赠金金额，需平掉所有仓位，清空所有赠金 3.4 持仓中且收盘时，MT交易账号净值 小于等于 赠金金额，需等周一开始立马以开市价平仓，清空所有赠金 3.5 未持仓且开市时， MT交易账号净值 小于等于 赠金金额，清空所有赠金 3.6 未持仓且收盘时， MT交易账号净值 小于等于 赠金金额，清空所有赠金 3.7 PAMM投资，资金产生变动，清空所有赠金 **4. 赠金比例：** 4.1 首次入金赠金比例为60%，首次的维度是MT交易账号的维度，一个平台账户可以创建多个MT交易账号 ； 后续入金的赠金比例为30% ，一个平台账户 赠金封顶3万，封顶累计包含清空的 **5. 赠金到账规则：** 5.1 触发赠金自动审核通过，赠送到入金的MT账号 **6. 赠金异常情况：** 6.1 如果MT服务器掉线，无法链接
